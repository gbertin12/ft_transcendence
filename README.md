dumb example for Nest.js backend + DB

## Create user

```bash
curl localhost:3000/auth/signup -d 'username=admin&password=admin'
```

## Login

```bash
curl localhost:3000/auth/login -d 'username=admin&password=admin'
```

## Get all users

```bash
curl localhost:3000/auth/all
```
